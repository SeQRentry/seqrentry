SeQRentry.NET

Secure web accounts

Secure ... what?

SeQRentry is a highly secure, QR-based method to sign in to web sites online by using your cell phone's camera. The phone app automatically generates unique, cryptographically strong passwords for every site you enable — SeQRentry passwords are never reused across different accounts. Yet, the only thing you need to remember in order to log in is your phone and a personal PIN code.

Any web site that uses a username and password can be SeQRentry-enabled within minutes. The system also works well with the browser's built-in ability to remember account names and passwords. Strong, unique passwords on the Internet have never been easier.
